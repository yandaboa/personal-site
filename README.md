# Personal Website (Research + SWE)

Created in `personal-site/`.

## Run locally

From this folder:

```bash
python -m http.server 8000
```

Then open: `http://localhost:8000`

## Customize fast

- Edit content data in `site-data.js` (publications, experience, projects)
- Edit name/contact in `index.html`
- Adjust look in `styles.css`

## Deployment options

- GitHub Pages
- Netlify
- Vercel

This template is publication-forward while still highlighting SWE experience.
